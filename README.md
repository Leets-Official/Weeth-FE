## 📱프로젝트 소개  
  
  
**동아리는 우리 모두가 함께 하는 것! with, Weeth!**  
  
Weeth는 가천대학교 IT 학술동아리 Leets를 위한 동아리 관리 서비스 입니다.  
<br>일정, 출석체크, 공지사항, 회비 등 동아리 활동에 필요한 여러 편의 기능을 담았습니다.  
<br>함께라는 가치를 통해 동아리를 더욱 즐겁게!  
<br>Weeth를 통해 성장하는 Leets를 만들어가요🌟
